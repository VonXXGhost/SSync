# SSync

同步指定文件夹内容到目标文件夹。

### Options

```
--configFile, -c [] -> 配置文件，文件编码应为UTF-8 { String }
--checkModel, -cm -> 检查模式 { Value should be one of [simple, checksum] }
--srcPath, -s -> 源路径 { String }
--destPath, -d -> 目标路径 { String }
--preview, -p -> 是否仅预览检查结果 
--include, -in -> 白名单绝对路径正则 { String }
--exclude, -ex -> 排除绝对路径正则 { String }
--recursive, -r -> 是否递归子文件夹 
--help, -h -> Usage info
```
